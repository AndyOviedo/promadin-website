# Fundación Promadin - Sitio Web

Sitio web moderno y responsivo para la Fundación Promadin, una organización sin ánimo de lucro dedica a la educación e inclusión social.

## Características

- **Diseño Moderno**: Interfaz limpia y profesional con animaciones fluidas
- **Responsive**: Totalmente adaptado para dispositivos móviles, tablets y escritorio
- **Componentes Reutilizables**: Arquitectura modular con React
- **Herramientas Modernas**: Construido con Vite, React 18 y Tailwind CSS
- **Iconos Premium**: Librería lucide-react para iconografía consistente

## Requisitos Previos

- **Node.js** 16.0 o superior
- **npm** o **yarn** (gestor de paquetes)

## Instalación

1. **Navega al directorio del proyecto:**
   ```bash
   cd PromaDesign
   ```

2. **Instala las dependencias:**
   ```bash
   npm install
   ```

## Comandos Disponibles

### Desarrollo
Inicia el servidor de desarrollo en `http://localhost:3000`:
```bash
npm run dev
```

### Build para Producción
Construye el proyecto optimizado para producción:
```bash
npm run build
```

### Preview de Producción
Visualiza la versión de producción localmente:
```bash
npm run preview
```

## Estructura del Proyecto

```
PromaDesign/
├── src/
│   ├── App.jsx          # Componente principal
│   ├── main.jsx         # Punto de entrada
│   └── index.css        # Estilos globales
├── index.html           # Archivo HTML base
├── vite.config.js       # Configuración de Vite
├── tailwind.config.js   # Configuración de Tailwind CSS
├── postcss.config.js    # Configuración de PostCSS
└── package.json         # Dependencias y scripts
```

## Tecnologías Utilizadas

- **React 18** - Biblioteca para construir interfaces
- **Vite** - Bundler moderno y rápido
- **Tailwind CSS** - Framework de estilos utilitarios
- **lucide-react** - Librería de iconos
- **PostCSS** - Herramienta de transformación de CSS

## Secciones del Sitio

1. **Inicio** - Hero section con presentación de la fundación
2. **Sobre Nosotros** - Misión, Visión y Propósito
3. **Talleres** - Programas y actividades disponibles
4. **Contacto** - Información de contacto y ubicación
5. **WhatsApp CTA** - Botón flotante de contacto
6. **Footer** - Información legal y redes sociales
